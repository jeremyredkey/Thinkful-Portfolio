  • Loan Calculator App


    Github Repo
    Description: I was inspired by Credit Karma's application so I created an app like it without the Credit Score options dropdown selector. The purpose of this application was to test my abilities to manipulate the DOM with Javascript while creating a tool that can help financial loan officers. I learned many things from this project but the biggest help was once I found parseFloat() on MDN (this is particularly useful when dealing with money). Also I learned the formula on how to calculate loan total interest, monthly payment, and total payment by using this website: https://www.calculatorsoup.com
